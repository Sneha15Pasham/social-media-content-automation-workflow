# 🚀 Social Media Content Automation Workflow

## 📌 Project Overview

This project automates the creation and publishing of social media content using **n8n** and **Google Gemini AI**. It generates engaging LinkedIn posts from user prompts and publishes them automatically, reducing manual effort and ensuring consistent content delivery.

---

## ✨ Features

- AI-generated social media posts using Google Gemini
- Automated workflow built with n8n
- LinkedIn post automation
- Reduces manual effort
- Easy to customize for different content topics

---

## 🛠 Tech Stack

- n8n
- Google Gemini AI
- LinkedIn API
- JSON Workflow

---

## 🔄 Workflow

1. User provides content or a prompt.
2. Google Gemini generates a professional social media post.
3. The generated content is formatted.
4. The workflow publishes the post to LinkedIn automatically.

---

## 📂 Project Files

- `workflow.json` – Complete n8n workflow

---

## 🚀 How to Use

1. Download the workflow JSON file.
2. Open n8n.
3. Import the workflow.
4. Configure your API credentials.
5. Execute the workflow.
